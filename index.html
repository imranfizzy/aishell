AI Terminal

AI-Powered Command Line Interface

Run Grok-3, GPT-4o, Gemini-2.5-Flash, GLM-4.5-Air, Qwen2.5-VL-32B (Vision), TurboAI (ChatGPT-3), and SearchAI (Google AI) in your browser.

Use sudo- prefixed commands to control the AI.

Run Locally on Your PC (No Internet After First Load)
Paste this in PowerShell or CMD: powershell -Command "Invoke-WebRequest 'powershell -Command "Invoke-WebRequest 'https://imranfizzy.github.io/ai-shell/' -OutFile %TEMP%\ai.html; start %TEMP%\ai.html" → Opens AI Terminal instantly in your default browser.

Command Reference:

  • sudo-get/[your question] → Ask the current AI model
  • sudo-clear/ → Clear the screen
  • sudo-save/ → Download chat as .txt
  • sudo-color/[color] → Change background (red, blue, green, etc.)
  • sudo-gethis/ → Show full chat history
  • sudo-getmodel/nanoblue → Switch to Grok-3 (xAI)
  • sudo-getmodel/nanoblack → Switch to GPT-4o (OpenAI)
  • sudo-getmodel/tinynano → Switch to Gemini-2.5-Flash
  • sudo-getmodel/bashai → Switch to GLM-4.5-Air (Free)
  • sudo-getmodel/minyai → Switch to Qwen2.5-VL-32B (Vision)
  • sudo-getmodel/turboai → Switch to TurboAI (ChatGPT-3)
  • sudo-getmodel/searchai → Switch to SearchAI (Google AI)
